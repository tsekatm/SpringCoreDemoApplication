# SpringCoreDemoApplication

This Spring Boot application demonstrates core concepts including dependency injection, RESTful API design, and component scanning.

## Features
- Constructor injection for loose coupling
- RESTful endpoint for daily workout suggestions
- Utilizes Spring's component scanning

## How to Run
1. **Prerequisites:** Ensure you have Java and Maven installed on your system.
2. **Clone the repository:** `git clone https://github.com/tsekatm/SpringCoreDemoApplication.git`
3. **Navigate to the project directory:** `cd SpringCoreDemoApplication`
4. **Build the project:** `mvn clean install`
5. **Run the application:** `mvn spring-boot:run`
6. **Access the endpoint:** Open your web browser and navigate to `http://localhost:8080/dailyworkout`

## Technologies Used
- Java
- Spring Boot
- Maven
